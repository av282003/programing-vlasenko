Завдання 20. ( ** ) Підрахувати суму чисел у заданому діапазоні. Наприклад, при вхідних даних 50 та 52
повинно бути 50 + 51 + 52 = 153 .

Хід роботи:
Сумма чисел у заданому діапозоні визначаеться за формулаю S = (a+b)/2*(b - a + 1 ),де а та b діапозон(a < b )
Перелік вхідних данних:
-а та b - період (int)
-res- результат (double)
Дослідження результатів программи:
res - наш результат,тобто S.Для уникнення похибки ми вимущенні іспользувати тип int для интервалу,але через операцію  ділення ми повинні  результат визначити як double.
-при значенні a = 50,b = 52 res = S = (50+52) / 2 * (52 - 50 + 1) = 153
-для пітвердження коректності роботи програми,зупинемо відлагодникна стрці з return 0 та введемо команду fr v.Отримуем наступне:
(int) a = 50
(int) b = 52
(int) res = 153
